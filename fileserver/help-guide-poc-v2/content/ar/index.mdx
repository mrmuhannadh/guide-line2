# المقدمة

تم تصميم **Pro12** لمساعدتك في تحقيق أهدافك في التداول والاستثمار من خلال جميع أفضل الأدوات في منصة واحدة.

![Pro12](/home.png)

## المزايا [#features]

- أداء وسرعة ممتازان.
- بيانات السوق اللحظية، الأخبار، الإعلانات، والأدوات التحليلية.
- القوائم المالية الكاملة والنسب المالية.
- لوحات معلومات جاهزة للتنقل السريع والسهولة، مع إمكانية إنشاء لوحات ومناطق عمل مخصصة.
- مخططات احترافية متقدمة مع المزيد من دراسات الخطوط، المؤشرات، الاستراتيجيات الفنية، وباني معادلات شامل للمؤشرات المخصصة.
- تصفية قوائم المتابعة بناءً على معادلات مخصصة مع تنبيهات لحظية.
- متعقب التدفقات النقدية للقطاعات.
- المحلل الرقمي والماسح الفني.
- تحسين وظائف أحداث الشركات.
- تأثير السهم على مؤشر تاسي ومؤشرات القطاعات.
- ودجت الصفقات الكبيرة.

## متطلبات النظام [#system-requirements]

|                   |          **الحد الأدنى**           |                  **الأفضل** |
| :---------------- | :--------------------------------: | --------------------------: |
| المعالج (CPU)     | Intel Core i3 أو معالج بأربع أنوية |       Intel Core i5 أو أعلى |
| الذاكرة (RAM)     |             8 جيجابايت             |         16 جيجابايت أو أكثر |
| اتصال الإنترنت    |          2.0 ميجابت/ثانية          |      2 ميجابت/ثانية أو أعلى |
| نظام التشغيل (OS) |             Windows 10             | Windows 11 x64 (أحدث إصدار) |
