# Conditional Dashboard

<FigureProvider>

View all Conditional Orders with filtered order statuses (Open/Today's, Executed, Rejected, and Cancelled), including Child Conditional Orders, new requests, and searching conditional orders via this window. You can access this window via the Trade Dashboard or via the Menu.

### Window invoking options

1. Trade Dashboard → Conditional Orders
2. Menu → Trade → Conditional Orders Dashboard
