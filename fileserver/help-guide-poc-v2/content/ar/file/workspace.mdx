# Workspace

<FigureProvider>

You may save your workspace with the current settings to open it when you log in to the system next time.

## Save Workspace

### Option 1

**STEPS**

1. To save the current state or settings of the working workspace,
2. Navigate to the **Save Workspace** section under **File** section.
3. Click on the **Save Workspace**
4. The current workspace successfully get saved

<Figure
  baseName='workspace/save-workspace'
  alt='save-workspace'
  description='Save Workspace'
/>

### Option 2

**STEPS**

1. To save the current workspace click **CTRL + S**
2. The current workspace successfully get saved

## Save Workspace As Different Name

### Option 1

**STEPS**

1. To save the current state or settings of the working workspace, click on the
2. Navigate to the **Save Workspace As** section under **File** section.
3. Click on the **Save Workspace As**

<Figure
  baseName='workspace/save-workspace-as'
  alt='save-workspace-as'
  description='Save Workspace As'
/>

4. This prompts a dialog box you to enter a new name for the workspace with 3 options namely, **Save as Default**, **Save** and **Cancel**

<Figure
  baseName='workspace/save-workspace-dialog'
  alt='save-workspace-dialog'
  description='Save Workspace dialog box'
/>

5. A name for the workspace can be entered in Workspace Name which is a mandatory field
6. If you want to save as default to open it in the first place once log in to the system,

   6.1. Click on the **Save As Default** button in the prompt to save this workspace as the default workspace.

7. Click on **Save** if you wish to only save, but not as the default workspace that opens when logged in.

### Option 2

**STEPS**

1. To save the current workspace click **CTRL + SHIFT + S**
2. This prompts a dialog box you to enter a new name for the workspace with 3 options namely, **Save as Default**, **Save** and **Cancel**
3. A name for the workspace can be entered in Workspace Name which is a mandatory field
4. If you want to save as default to open it in the first place once log in to the system,

   4.1. Click on the **Save As Default** button in the prompt to save this workspace as the default workspace.

5. Click on **Save** if you wish to only save, but not as the default workspace that opens when logged in.
