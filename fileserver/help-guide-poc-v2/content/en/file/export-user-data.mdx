This section describes how to export user data.

## Export User Data

**STEPS**

1. Navigate to the File section on the left top hand corner of the system
2. Expand the File section and navigates to the **Export User Data** section under File as given below,

<Figure
  baseName='import-export-user-data/export-via-file'
  alt='export-via-file'
  description='Export User Data via File'
/>

3. Click on Export User Data
4. If the workspace is not saved it prompts the below dialog box to save the workspace
5. If the user select Yes and if the workspace is not saved it prompts the below dialog box to save the workspace

<Figure
  baseName='import-export-user-data/save-workspace-dialog'
  alt='save-workspace-dialog'
  description='Export User Data save workspace Dialog box'
/>

6. When the user clicks the **Export User Data** it downloads a DAT file
