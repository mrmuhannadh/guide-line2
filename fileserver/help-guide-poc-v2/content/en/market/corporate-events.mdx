## Filter Corporate Events

- **Filter by Category** – Click on the **Category** drop down and select a category to filter details from. Each category will be color coded.
- **Filter by Symbol/Event** – Click on the **Symbol/Event** text box, and type a symbol name or the event name to filter corporate events.

## Search Corporate Events

Search a Corporate Event within a date range by clicking on the **From** and **To** date pickers to pick a date range, and click on the **Search** button

## Corporate Event Calendar

1. The Corporate Event Calendar will display the event dates in colors, and each color will represent an event category type.
2. Scrolling the mouse over the calendar date that is colored, will pop up summary details of the event as given below,
3. Click on the year on top, to load that year's calendar as given below,
