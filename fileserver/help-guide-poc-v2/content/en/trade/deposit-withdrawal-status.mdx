# Deposit/ Withdrawal Status

<FigureProvider>

View all deposit/withdrawal statuses via this window. You can access the Deposit/Withdrawal Status window via the Menu.

### Window invoking options

1. Trade → Deposit/Withdrawal Status
2. Portfolio → Account Summary → Deposit/Withdrawal Status
3. Order List → Right Click → Deposit/Withdrawal Status

The Deposit/Withdrawal status can be directly found via Account Summary section as given,

## Search

STEPS

1. Select Customer name
2. Select a From and To date from the date pickers.
3. Click on the Search button to search and display portfolios.

## Filters

- **Customer ID** – Filter details by customer ID by entering the customer ID.
- **Date** – Filter details by date range From to TO
- **Type** – Filter details by deposit, withdrawal, transfer, or all types.
- **Status** – Filter details by a status.
